# Monkey API

Testing Android API in randomized manner. Mozilla has done this to find out their bugs in the name of _Lang Fuzz_, coined by a guy named *Christian Holler*. Here’s his paper, presentation, and online book.  

- [The Fuzzing Book : Tools and Techniques for Generating Software Tests](https://www.fuzzingbook.org/)  
- [Dos And Don'ts When Introducing New Fuzzing Tools](https://www.code-intelligence.com/blog/implementing-fuzzing-tools)  
- [Fuzzing with Code Fragments](https://www.st.cs.uni-saarland.de/publications/files/holler-usenix-2012.pdf)  
- <https://www.youtube.com/watch?v=LDVfLX2KdVM>
